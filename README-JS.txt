1. Навигационные меню по товарам в хедере отстраиваются классом js/NavRender.js по шаблону структуры меню data/dto.js:
	- brouse - по меню 2-го уровня
	- nav - по меню 3-го уровня

2. Все страницы с товарами к покупке отрисовывают блоки с товарами классом js/GoodsView.js по алгоритму:
	1. Получить список товаров с сервера (сейчас силами json-server db.json)
	2. Отфильтровать полученный список по html атрибуту'data-filter'
	3. Отрисовать отфильтрованные товары на странице: 
		index.html - нет фильтра, отрисовывает все товары
		product-catalog-men.html - фильтр по свойству {category:men}
		single-good-card.html - фильтр по свойству {productType: } в зависимости от основного товара страницы 

3. Управление корзиной js/Cart.js; прием и удаление товаров силами json-server db.json
	если в корзине +=2 товара - добавляет эффект анимации

4. rangeSlider.js - добавление эффекта jQuerryUI для отображения линейки фильтра товаров по цене для всех страниц product.catalog.html
